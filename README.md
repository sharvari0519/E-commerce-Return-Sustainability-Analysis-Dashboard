# E-commerce Return & Sustainability Analysis Dashboard

Built an interactive E-commerce Return & Sustainability Dashboard using Power BI, MySQL, Excel, Power Query, and DAX. Analyzed 5,000+ order records to track return trends, revenue loss, return reasons, customer locations, and sustainability metrics through dynamic visualizations and interactive dashboards.

## Project Structure

- **`ecommerce.pbix`**: Power BI Desktop dashboard file providing interactive visualizations of key performance indicators (KPIs), return rates, product category performance, and sustainability insights.
- **`ecommerce.pdf`**: PDF export of the Power BI dashboard showing the visual layouts and key reports.
- **`eccommerce.sql`**: SQL script containing data extraction, cleaning, aggregation, and analysis queries (e.g., net revenue calculations, monthly/yearly trend analysis, customer segmentation, and order quantity analytics).
- **`returns_sustainability_dataset.csv`**: The primary dataset containing transactions, product details, return statuses, and customer demographics used for this analysis.

## Key Insights & Features Analyzed

- **Revenue & Sales Performance**: Total and average order value, net revenue, monthly and annual sales trends.
- **Product Analysis**: Category-wise revenue contribution, top-selling items, and pricing structures.
- **Customer Insights**: Identification of high-value customers and purchase frequencies.
- **Transactional Metrics**: Payment method distribution and average purchase quantities.
